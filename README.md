## 学習コンテンツ

### Step 01: GitHub Copilot

GitHub Copilot の基本的な機能と使用方法を学ぶためのサンプルアプリケーションです。

**内容**: Next.js + TypeScript による Todo リストアプリケーション

**学習ポイント**:
- コード補完と型推論
- コンポーネント生成パターン
- コメントからのコード生成
- リファクタリング支援

**技術スタック**: Next.js 16.1.1, React 19.2.3, TypeScript, Tailwind CSS 4

詳細は [step01_github_copilot/README.md](step01_github_copilot/README.md) をご参照ください。
